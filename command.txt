GEW Intelligence OS - Root Command Guide

Use these commands from the project root: P:\Gew

python run_current_phases.py
Runs the FULL pipeline (Phases 1-5) sequentially:
1. CRM Mapping & Transcription
2. Voice Intelligence (Acoustics/Emotion)
3. AI Strategic Reasoning
4. Knowledge Layer (Vector Indexing)
5. Query Intelligence (Decision Engine)

python run_current_phases.py --call-id CALL_0006
Runs the FULL pipeline for a single specific audio file only.

python main.py
Runs Phase 1: Ingestion & Transcription only.

python -m phase_2_enrichment_structured_extraction.main
Runs Phase 2: Voice Intelligence (Acoustics & Emotion enrichment).

python -m phase_3_ai_reasoning.main
Runs Phase 3: AI Reasoning and Strategic analysis.

python -m phase_4_knowledge_memory.main
Runs Phase 4: Knowledge indexing and memory building.

python -m phase_5_query_intelligence.query_main --query "YOUR QUESTION"
Runs a specific query against the processed knowledge base.

python api.py
Starts the FastAPI server on port 8000 for the web frontend or external integrations.

run_background.bat
Starts the pipeline in a minimized background window.

pip install -r requirements.txt
Installs the necessary Python dependencies.
